Git & GitHub Glossary

Repository (Repo)

A storage space where your project lives, containing all files and their version history.

Commit

A snapshot of your repository at a specific point in time with a descriptive message.

Branch

A parallel version of a repository that allows you to work freely without affecting the main project.

Merge

The process of combining changes from one branch into another branch.

Pull Request

A method of submitting contributions to a project, requesting that changes be reviewed and merged.

Clone

Creating a local copy of a remote repository on your computer.
